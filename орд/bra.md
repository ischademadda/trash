


Вот техническое описание пользовательского интерфейса (UI) представленного экрана, структурированное по функциональным блокам (сверху вниз). Описание сфокусировано на элементах управления и их состояниях.

### 1. Навигация и заголовок
*   **Кнопка «Назад»:** Текстовая ссылка с иконкой (шеврон влево). Функция — возврат на предыдущий экран.
*   **Заголовок:** H1 «Регистрация креатива».

### 2. Индикатор прогресса (Stepper)
Состоит из трех шагов. Выполняет функцию отображения текущего этапа процесса.
*   **Шаг 1:** «Договор». Состояние: *Завершен* (светло-синий фон, иконка галочки, синий текст). Предположительно кликабелен для возврата.
*   **Шаг 2:** «Информация о креативе». Состояние: *Активен* (желтый фон с цифрой 2, черный текст). Текущий экран.
*   **Шаг 3:** «Материалы креатива». Состояние: *Неактивен/Заблокирован* (серый фон с цифрой 3, серый текст).

### 3. Основная форма (Form Fields)
Каждое текстовое поле (input) имеет стандартизированный дизайн: скругленные углы, серый фон. Большинство полей имеют встроенные экшены: `Clear` (крестик) и `Tooltip/Help` (вопросительный знак).

*   **Поле 1: Название кампании (Text Input)**
    *   *Лейбл:* «Название для рекламной кампании» (внутри поля, сверху).
    *   *Значение:* Введен текст.
    *   *Контролы внутри поля:* Кнопка очистки поля (крестик), иконка подсказки (вопрос).

*   **Поле 2: Категория/ОКВЭД (Tag Input / Multi-select)**
    *   *Лейбл:* Отсутствует.
    *   *Значение:* Выбран элемент в виде тега («29.2.1. Банковские вклады»). У самого тега есть внутренний крестик для удаления конкретного выбора.
    *   *Контролы внутри поля (справа):* Кнопка очистки всего поля (крестик), иконка подсказки.

*   **Поле 3: Период (Date Range Picker)**
    *   *Лейбл:* «Период проведения».
    *   *Значение:* Диапазон дат (ДД.ММ.ГГГГ – ДД.ММ.ГГГГ).
    *   *Контролы внутри поля:* Кнопка очистки (крестик), иконка вызова календаря (справа).

*   **Блок 4: Модель оплаты и Формат (Два поля в одну строку)**
    *   **Левое поле (Dropdown/Select):** Выбор модели.
        *   *Лейбл:* Отсутствует (текущее значение «CPM» выглядит как placeholder или выбранное значение).
        *   *Контролы:* Очистка (крестик), шеврон раскрытия списка (вниз).
    *   **Правое поле (Dropdown/Select):** Выбор формата.
        *   *Лейбл:* Отсутствует.
        *   *Значение:* «Видеоролик».
        *   *Контролы:* Очистка (крестик), шеврон раскрытия списка (вниз).

*   **Блок 5: Целевые ссылки (Dynamic Input Group)**
    *   *Группа состоит из поля ввода и отдельной кнопки удаления.*
    *   **Поле (Text Input):**
        *   *Лейбл:* «Целевая ссылка рекламной кампании».
        *   *Значение:* URL-адрес.
        *   *Контролы:* Очистка (крестик), иконка подсказки.
    *   **Кнопка удаления:** Иконка корзины в отдельном сером квадрате справа от поля. Удаляет текущую строку с ссылкой.
    *   **Экшен добавления:** Текстовая кнопка «Добавить еще» (синяя, под полем ввода). Функция — динамическое добавление нового пустого поля для ввода дополнительной ссылки.

*   **Поле 6: Описание (Textarea)**
    *   *Лейбл:* «Описание рекламной кампании».
    *   *Значение:* Многострочный текст.
    *   *Контролы внутри поля:* Иконка подсказки (в правом верхнем углу).
    *   *Валидация/Счетчик:* Счетчик символов в правом нижнем углу («37/200»). Ограничение длины ввода — 200 символов.

### 4. Дополнительные параметры (Checkboxes)
Расположены в строку. Квадратные чекбоксы без рамки (серый фон).
*   **Чекбокс 1:** «Социальная реклама». Состояние: *Не выбран* (unchecked).
*   **Чекбокс 2:** «Социальная реклама по квоте». Состояние: *Не выбран* (unchecked).

### 5. Кнопки действий (Action Buttons)
*   **Primary Button:** «Продолжить». Кнопка с заливкой (желтая). Сохраняет данные и переводит на Шаг 3.
*   **Secondary/Ghost Button:** «Отменить». Только синий текст без фона. Прерывает процесс/закрывает форму.
